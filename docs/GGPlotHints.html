GGPlot Snippets

This is a collection of plots I find myself making often.

Scatter plot

data(CO2)
CO2 %>% 
  ggplot(aes(conc,uptake)) +
  geom_point()

Plot of regression coefficients

library(coefplot)
data("mtcars")
m1 = glm(am == 1 ~ mpg + cyl + disp  + wt, data = mtcars, family = "binomial") %>% 
  coefplot() %>% 
  print()
